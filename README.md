# الصرح للخدمات القانونية | Alsarh Legal Services

موقع فرونت-أند متطور لشركة محاماة سعودية متخصصة في القضاء الإداري. تم تصميمه بمعايير جمالية عالية (Premium Aesthetics) ليعكس القوة والمهنية والنزاهة.

## التقنيات المستخدمة
- **React + Vite**: لسرعة وأداء فائقين.
- **Vanilla CSS**: للتحكم الكامل في التصميم (Design Tokens).
- **Framer Motion**: للمسّات الجمالية والتحركات السلسة.
- **Lucide React**: للأيقونات العصرية.
- **Docker**: جاهز للرفع المباشر على Coolify.

## المميزات
- **تصميم إقليمي (Saudi Modern)**: يجمع بين الأصالة وحداثة الأنظمة السعودية.
- **تجربة مستخدم متميزة**: تنقل سلس بين الصفحات (Home, About, Services, Blog, Contact).
- **Glassmorphism**: نظام ملاحة (Navigation) شفاف وعصري.
- **Responsive Design**: متوافق تماماً مع الجوال والأجهزة اللوحية.
- **رقم تواصل مخصص**: يبدأ بـ 9002 كما هو مطلوب.

## التشغيل المحلي
```bash
npm install
npm run dev
```

## الرفع والتشغيل (Coolify / Docker)
المشروع يحتوي على `Dockerfile` جاهز. يمكنك ببساطة ربط المستودع بـ Coolify وسيتعرف عليه تلقائياً لبدء البناء والنشر.
